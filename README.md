## Project Setup

### Clone the repository
```bash
git clone https://github.com/ganeshku73/retailer_int.git
```

## Install dependencies

```bash
npm install
```
## Start run project
```bash
npm run start
```

## To check test cases run 
```bash
npm run test
```
